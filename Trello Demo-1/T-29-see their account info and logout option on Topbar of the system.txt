Screen Definition:

Logo
Empty section
Logged-in user name
Logout button

User Stories:

As a user, I want to see my account info and logout option so that I can manage my session easily.

Acceptance Criteria:

• Logo should be visible on the top bar.
• Logged-in user name should be displayed.
• Logout option should be available.
• Logout should erase session cookies.
• User should not be able to access dashboard without logging in again.
• User should be redirected to login page after logout.
• Top bar should be visible on all pages after login.
