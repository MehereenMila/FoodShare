User Stories:

As an Admin, I want to view a centralized, management dashboard with a access of structured navigation menu so that I can quickly monitor user oversight, system status, task assignments. Also I want a high-level overview cards so that I can monitor system health at a glance and navigate quickly to pending tasks.

Acceptance Criteria-

• Dashboard should load overview card by default and statistics should reflect latest data.
• Dashboard should be restricted to data visualization. No crud operation should be initiated directly on the dashboard
• The role based menu must be easily accessible from the main dashboard for users with "Admin" role
• Menu items for "Pending Verifications" or "Reported Issues" should display a numerical badge (such as- a red badge "5") indicating items requiring immediate action.
• Each sidebar link should successfully redirect the user to the associated functional page.
