User Stories:
As a user, I want the platform to be user-friendly to use so that I can complete my tasks quickly without confusion.

Acceptance Criteria:
• The interface should have clear labels and simple navigation.
• Buttons and actions should be easy to find and understand.
• All pages should follow a consistent and clean layout.
• Users should be able to complete basic tasks without confusion or extra help.
• Error messages should be clear and easy to understand.
• The system should be easy to use on both desktop and mobile devices.
