Screen Definition:
• Email field
• Send Code button field
• OTP Code field
• New Password field
• Confirm Password field
• Submit button field

User Story:
As a registered user, I want to get back my account whenever I forgot my password I can reset it by forget password button.

Acceptance Criteria-
• The Email field is mandatory. If left empty and the Send Code button is clicked, the system should display "Email is required."
• The system should verify whether the entered email exists in the database. If the email does not exist, the system should display "Email not found." Otherwise, it should display "Email found."
• Clicking the Send Code button should send an OTP to the registered email address and should display "OTP sent."
• The OTP field is mandatory. If left empty and the Check button is clicked, the system should display "OTP is required."
• When the Check button is clicked, the system should validate the entered OTP. If the OTP is invalid or expired, the system should display "Invalid or expired OTP." If valid, the system should make the New Password, Confirm Password, and Confirm Changes button visible.
• If the new password does not meet the criteria, the system should display "Password does not meet criteria."
• The Confirm Password field is mandatory and must match the new password; otherwise, the system should display "Password does not match."
• Clicking the Confirm button should update the user’s password in the database.
• After a successful update, the system should display "Password successfully changed." and redirect the user to the Login page.
